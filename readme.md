About
================================================================================

An ansible playbook for configuring different kinds of servers. Including roles
for:

- Basic server
- Apache web server
- MySQL db server
- Rsnapshot backup server
- Nagios monitoring server

Tested on CentOS 6.5. Works with SELinux enabled.
